# Shop-java-project <br>
Java Project for University  <br>
full project on Java GUI <br>

This Java project involves creating a store management system with a Graphical User Interface (GUI) using the Swing GUI toolkit. The system will allow users to manage employees, customers, products, sales, inventory, and billing. The goal of this system is to help store owners manage their store more efficiently and improve their sales and customer experience. Thos project is fully based on Java and there is no use of Database, we use .txt file for storing and viewing data.
